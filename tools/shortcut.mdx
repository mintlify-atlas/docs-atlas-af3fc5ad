The **Shortcut** tool executes keyboard shortcuts by pressing key combinations. Supports all standard Windows shortcuts including modifier keys (Ctrl, Alt, Shift, Win) and special keys.

## Purpose & Use Cases

- **Quick Actions**: Copy, paste, save, undo, redo
- **Window Management**: Switch apps, minimize, maximize, close
- **System Commands**: Open Start menu, Task Manager, Run dialog
- **Application Shortcuts**: Open menus, toggle features, navigation
- **Text Editing**: Select all, find, replace, formatting
- **Clipboard Operations**: Cut, copy, paste

## Parameters

<ParamField path="shortcut" type="string" required>
  Keyboard shortcut using `+` to join keys. Format: `"modifier+modifier+key"`
  
  Examples:
  - `"ctrl+c"` - Copy
  - `"ctrl+shift+esc"` - Task Manager
  - `"win+r"` - Run dialog
  - `"alt+f4"` - Close window
  - `"win"` - Start menu (single key)
</ParamField>

## Return Values

Returns a success message:
```
"Pressed ctrl+c."
"Pressed win+r."
```

## Supported Keys

### Modifier Keys
- `ctrl` / `control` - Control key
- `alt` / `option` - Alt key (Option on Mac-style keyboards)
- `shift` - Shift key
- `win` / `windows` / `command` - Windows/Command key

### Special Keys
- `enter` - Enter/Return
- `tab` - Tab
- `esc` / `escape` - Escape
- `space` / `spacebar` - Space
- `backspace` / `back` - Backspace
- `delete` / `del` - Delete
- `home` - Home
- `end` - End
- `pageup` / `pgup` - Page Up
- `pagedown` / `pgdn` - Page Down
- `insert` - Insert
- `capslock` / `capital` - Caps Lock
- `scrolllock` / `scroll` - Scroll Lock

### Arrow Keys
- `left` / `leftarrow` - Left arrow
- `right` / `rightarrow` - Right arrow
- `up` / `uparrow` - Up arrow
- `down` / `downarrow` - Down arrow

### Function Keys
- `f1` through `f12` - Function keys
- `f13` through `f24` - Extended function keys (if available)

### Character Keys
- `a` through `z` - Letter keys
- `0` through `9` - Number keys
- Special characters: `-`, `=`, `[`, `]`, `;`, `'`, `,`, `.`, `/`, `\`

## Best Practices

<Tip>
**Wait After Shortcuts**
Always add a small wait after shortcuts that trigger UI changes:
```python
Shortcut("ctrl+o")  # Open file dialog
Wait(duration=0.5)  # Wait for dialog to appear
Snapshot()  # Capture new state
```
</Tip>

<Tip>
**Use Shortcuts Over Clicking**
Shortcuts are often faster and more reliable than clicking menu items:
```python
# BETTER: Direct shortcut
Shortcut("ctrl+s")

# SLOWER: Click through menu
Click(label=1)  # File menu
Wait(duration=0.3)
Click(label=5)  # Save option
```
</Tip>

<Tip>
**Case Insensitive**
Key names are case-insensitive:
```python
Shortcut("CTRL+C")  # Same as
Shortcut("ctrl+c")  # Same as
Shortcut("Ctrl+C")  # All work
```
</Tip>

<Warning>
**Application-Specific Shortcuts**
Some shortcuts only work in specific applications:
- `ctrl+shift+t` - Browser only (reopen tab)
- `ctrl+k` - Different meaning in different apps
- `f2` - Rename in Explorer, Edit in Excel

Verify shortcut behavior for target application.
</Warning>

<Warning>
**Focus Requirements**
Shortcuts are sent to the **active window**:
- Ensure target window is active before sending shortcut
- Use `App(mode="switch")` to activate window first
- Some system shortcuts (Win+L, Ctrl+Shift+Esc) work globally
</Warning>

## Advanced Usage

### Chain Multiple Shortcuts
```python
# Select paragraph and format
Shortcut("ctrl+shift+down")  # Select paragraph
Wait(duration=0.2)
Shortcut("ctrl+b")  # Bold
Wait(duration=0.1)
Shortcut("ctrl+i")  # Italic
```

### Conditional Shortcuts
```python
# Different shortcuts based on application
state = Snapshot()
if "Chrome" in state.active_window:
    Shortcut("ctrl+t")  # New tab
elif "Notepad" in state.active_window:
    Shortcut("ctrl+n")  # New file
```

### Navigate Without Mouse
```python
# Keyboard-only navigation
Shortcut("tab")  # Move to next control
Wait(duration=0.2)
Shortcut("tab")
Wait(duration=0.2)
Shortcut("enter")  # Activate focused control
```

### Macro-Like Sequences
```python
# Complex multi-step operation
def save_and_export():
    Shortcut("ctrl+s")  # Save
    Wait(duration=1)
    Shortcut("alt+f")  # File menu
    Wait(duration=0.3)
    Type(text="e")  # Export option
    Wait(duration=0.5)
    Shortcut("enter")  # Confirm

save_and_export()
```

## Implementation Details

From `desktop/service.py:675-686`:

```python
def shortcut(self, shortcut: str):
    keys = shortcut.split("+")
    sendkeys_str = ""
    for key in keys:
        key = key.strip()
        if len(key) == 1:
            sendkeys_str += key
        else:
            name = _KEY_ALIASES.get(key.lower(), key)
            sendkeys_str += "{" + name + "}"
    uia.SendKeys(sendkeys_str, interval=0.01)
```

- Splits shortcut string by `+` separator
- Applies key aliases (e.g., `win` → `Win`)
- Single-character keys sent directly
- Multi-character keys wrapped in braces for UIAutomation
- 10ms interval between key presses
