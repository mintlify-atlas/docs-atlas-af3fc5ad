## Tool Selection Guide

### For UI Automation
1. Always start with **Snapshot** to understand the current state
2. Use **Click**, **Type**, and **Scroll** for basic interactions
3. Use **Move** for hover effects or drag-and-drop
4. Use **Shortcut** for quick system commands

### For Batch Operations
- **MultiSelect**: Select multiple files, checkboxes, or items
- **MultiEdit**: Fill forms with multiple input fields

### For Application Control
- **App**: Launch and manage windows
- **Process**: Monitor and terminate processes
- **PowerShell**: Execute complex system commands

### For Data Extraction
- **Scrape**: Get web page content (HTTP or DOM-based)
- **Clipboard**: Transfer data between applications
- **Snapshot**: Extract UI element information

<Note>
**Best Practice**: Always call Snapshot before performing UI interactions to get current element coordinates and labels.
</Note>

## Common Patterns

### Basic Workflow
```python
# 1. Capture current state
snapshot = Snapshot(use_vision=True)

# 2. Interact with UI
Click(label=5)  # Click element with label 5
Type(text="Hello World", label=10)  # Type in element 10

# 3. Submit and wait
Shortcut("ctrl+enter")
Wait(duration=2)
```

### Form Filling
```python
# Efficient multi-field form filling
MultiEdit(labels=[
  [5, "John Doe"],
  [6, "john@example.com"],
  [7, "555-1234"]
])
```

### File Management
```python
# Launch file explorer and select multiple files
App(mode="launch", name="explorer")
Wait(duration=1)
Snapshot(use_vision=True)
MultiSelect(labels=[10, 11, 12], press_ctrl=True)
```

## Coordinate System

All tools use **virtual desktop coordinates** which span across all monitors. The coordinate space:
- Origin (0,0) is at the top-left of the primary monitor
- Multi-monitor setups use a unified coordinate system
- Snapshot provides the current coordinate space dimensions

## Error Handling

All tools return descriptive error messages:
- Invalid coordinates or labels
- Missing required parameters
- State errors (e.g., calling Click with label before Snapshot)
- System-level failures

<Warning>
Windows-MCP has **full system access** with no sandboxing. Tools like PowerShell and App can perform irreversible operations. Use with caution.
</Warning>
