## Overview

The Tree service (`tree/service.py`) captures the Windows UI Automation accessibility tree to discover interactive and scrollable elements. It performs multi-threaded UI traversal and supports both standard Windows applications and browser DOM extraction.

## Key Classes

## Data Models

### TreeState

Complete UI element tree state.

```python
@dataclass
class TreeState:
    status: bool = True
    root_node: TreeElementNode | None = None
    dom_node: ScrollElementNode | None = None
    interactive_nodes: list[TreeElementNode] = field(default_factory=list)
    scrollable_nodes: list[ScrollElementNode] = field(default_factory=list)
    dom_informative_nodes: list[TextElementNode] = field(default_factory=list)
```

<Expandable title="TreeState Fields">
  <ResponseField name="status" type="bool">
    `True` if all windows captured successfully, `False` if some failed
  </ResponseField>

  <ResponseField name="root_node" type="TreeElementNode">
    Desktop root node representing the entire screen
  </ResponseField>

  <ResponseField name="interactive_nodes" type="list[TreeElementNode]">
    All clickable/typable elements (buttons, inputs, links, etc.)
  </ResponseField>

  <ResponseField name="scrollable_nodes" type="list[ScrollElementNode]">
    Elements with scrolling capability and scroll position metadata
  </ResponseField>

  <ResponseField name="dom_node" type="ScrollElementNode | None">
    Browser document root with scroll information (if browser active)
  </ResponseField>

  <ResponseField name="dom_informative_nodes" type="list[TextElementNode]">
    Text content extracted from browser DOM
  </ResponseField>
</Expandable>

### TreeElementNode

Represents a single interactive UI element.

```python
@dataclass
class TreeElementNode:
    bounding_box: BoundingBox
    center: Center
    name: str = ""
    control_type: str = ""
    window_name: str = ""
    metadata: dict[str, Any] = field(default_factory=dict)
```

<Expandable title="Metadata Fields">
  - **has_focused** (bool): Whether element currently has keyboard focus
  - **shortcut** (str): Accelerator key if available (e.g., "Alt+F")
  - **help_text** (str): Accessibility help text
  - **value** (str): Current value for inputs/edits
  - **is_password** (bool): Whether input is password field
  - **toggle_state** (str): "on" or "off" for checkboxes
  - **expand_collapse_state** (str): "expanded", "collapsed", or "partially expanded"
  - **is_selected** (bool): Whether item is selected
</Expandable>

### ScrollElementNode

Represents a scrollable container with scroll state.


## Usage Examples

### Capture UI Tree

```python
from windows_mcp.desktop.service import Desktop

desktop = Desktop()
tree = desktop.tree

# Get all window handles
handles = desktop.get_controls_handles()
active_handle = 123456  # Current window
other_handles = list(handles - {active_handle})

# Capture tree
tree_state = tree.get_state(
    active_window_handle=active_handle,
    other_windows_handles=other_handles,
    use_dom=False
)

print(f"Status: {tree_state.status}")
print(f"Interactive elements: {len(tree_state.interactive_nodes)}")
print(f"Scrollable elements: {len(tree_state.scrollable_nodes)}")
```

### Process Interactive Elements

```python
for idx, node in enumerate(tree_state.interactive_nodes):
    print(f"[{idx}] {node.control_type}: {node.name}")
    print(f"    Location: ({node.center.x}, {node.center.y})")
    print(f"    Window: {node.window_name}")
    
    if node.metadata.get("has_focused"):
        print("    [FOCUSED]")
    
    if "value" in node.metadata:
        print(f"    Value: {node.metadata['value']}")
```

### Browser DOM Extraction

```python
# Capture browser DOM instead of UI elements
tree_state = tree.get_state(
    active_window_handle=browser_handle,
    other_windows_handles=[],
    use_dom=True  # Extract DOM elements
)

if tree_state.dom_node:
    dom = tree_state.dom_node
    print(f"Vertical scroll: {dom.metadata['vertical_scroll_percent']}%")
    
    # Extract text content
    for text_node in tree_state.dom_informative_nodes:
        print(text_node.text)
```

### Check Scroll State

```python
for idx, scroll_node in enumerate(tree_state.scrollable_nodes):
    meta = scroll_node.metadata
    if meta.get("vertical_scrollable"):
        percent = meta.get("vertical_scroll_percent", 0)
        print(f"[{idx}] {scroll_node.name}: {percent}% scrolled")
        
        if percent < 100:
            print("  → Can scroll down")
```

## Performance Considerations

- **Sequential Processing**: Windows are processed sequentially to avoid COM apartment threading deadlocks (tree/service.py:85)
- **Retry Logic**: Failed window captures are retried up to 3 times with exponential backoff (tree/service.py:110)
- **Caching**: Uses UI Automation cache requests to minimize property access overhead (tree/service.py:580)
- **Clipping**: Bounding boxes are clipped to screen/window bounds to filter out-of-view elements (tree/service.py:140)
