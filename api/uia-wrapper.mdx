## Overview

The `uia/` module provides a Python wrapper around the Windows UI Automation COM API via `comtypes`. It abstracts the low-level COM interface for discovering, inspecting, and interacting with Windows UI elements.

Based on [Python-UIAutomation-for-Windows](https://github.com/yinkaisheng/Python-UIAutomation-for-Windows) by yinkaisheng.

## Architecture

The UIAutomation wrapper consists of four key modules:

- **core.py**: Main automation client and coordinate helpers
- **controls.py**: Control classes wrapping IUIAutomationElement
- **patterns.py**: UI Automation pattern interfaces (Invoke, Value, Scroll, etc.)
- **enums.py**: COM enumerations (ControlType, PatternId, PropertyId, etc.)
- **events.py**: Event subscription handlers

## UI Automation Patterns

Patterns provide specialized functionality for controls. Access via `GetPattern()`.

### InvokePattern

Invoke buttons/menu items.

```python
invoke = control.GetPattern(PatternId.InvokePattern)
invoke.Invoke()
```

### ValuePattern

Get/set control values.

```python
value = control.GetPattern(PatternId.ValuePattern)
value.SetValue("text")
current = value.Value
is_readonly = value.IsReadOnly
```

### ScrollPattern

Control scrolling.

```python
scroll = control.GetPattern(PatternId.ScrollPattern)
scroll.Scroll(ScrollAmount.LargeIncrement, ScrollAmount.NoAmount)

# Properties
scroll.HorizontallyScrollable  # bool
scroll.VerticallyScrollable    # bool
scroll.HorizontalScrollPercent # float (0-100)
scroll.VerticalScrollPercent   # float (0-100)
```

### TogglePattern

Toggle checkboxes.

```python
toggle = control.GetPattern(PatternId.TogglePattern)
toggle.Toggle()
state = toggle.ToggleState  # ToggleState.On, Off, Indeterminate
```

### WindowPattern

Window operations.

```python
window = control.GetPattern(PatternId.WindowPattern)
window.Minimize()
window.Maximize()
window.Close()

can_minimize = window.CanMinimize
can_maximize = window.CanMaximize
state = window.WindowVisualState  # Normal, Minimized, Maximized
```

### ExpandCollapsePattern

Expand/collapse tree items.

```python
expand = control.GetPattern(PatternId.ExpandCollapsePattern)
expand.Expand()
expand.Collapse()
state = expand.ExpandCollapseState  # Expanded, Collapsed, PartiallyExpanded
```

See `patterns.py` for complete list of patterns.

## Mouse & Keyboard Functions

### Mouse Control

```python
import windows_mcp.uia as uia

# Mouse movement
uia.MoveTo(x, y, moveSpeed=1)  # moveSpeed: 1=normal, <1=slower, >1=faster
uia.SetCursorPos(x, y)
uia.GetCursorPos() -> (x, y)

# Clicking
uia.Click(x, y, waitTime=0.5)
uia.RightClick(x, y)
uia.MiddleClick(x, y)

# Drag and drop
uia.DragDrop(x1, y1, x2, y2, moveSpeed=1)

# Scrolling
uia.WheelUp(wheelTimes=1)
uia.WheelDown(wheelTimes=1)
```

### Keyboard Control

```python
import windows_mcp.uia as uia

# Type text
uia.SendKeys("{Ctrl}a{Delete}Hello World{Enter}")

# Special keys
uia.SendKey(uia.Keys.VK_RETURN)
uia.PressKey(uia.Keys.VK_CONTROL)  # Hold
uia.ReleaseKey(uia.Keys.VK_CONTROL)  # Release

# Check key state
if uia.IsKeyPressed(uia.Keys.VK_SHIFT):
    print("Shift is pressed")
```

#### SendKeys Format

Special keys are enclosed in `{}`:

```python
uia.SendKeys("{Ctrl}c")           # Ctrl+C
uia.SendKeys("{Ctrl}{Shift}s")  # Ctrl+Shift+S
uia.SendKeys("{Win}r")           # Win+R
uia.SendKeys("{Ctrl}(AB)")      # Ctrl+A+B (hold Ctrl, press A, press B)
uia.SendKeys("{a 3}")            # Type 'a' 3 times
uia.SendKeys("{{Hello}}")        # Type '{Hello}'
```

Special key names: `{Ctrl}`, `{Shift}`, `{Alt}`, `{Win}`, `{Enter}`, `{Tab}`, `{Esc}`, `{Space}`, `{Back}`, `{Delete}`, `{Home}`, `{End}`, `{PageUp}`, `{PageDown}`, `{Left}`, `{Right}`, `{Up}`, `{Down}`, etc.

## Usage Examples

### Find and Click Button

```python
import windows_mcp.uia as uia

# Find button by name
button = uia.ButtonControl(Name="OK")
if button.Exists(maxSearchSeconds=5):
    button.Click()
```

### Type into Text Field

```python
import windows_mcp.uia as uia

# Find edit control
edit = uia.EditControl(AutomationId="searchBox")
if edit.Exists():
    # Clear and type
    value = edit.GetPattern(uia.PatternId.ValuePattern)
    value.SetValue("")
    edit.SendKeys("Hello World")
```

### Navigate Tree

```python
import windows_mcp.uia as uia

root = uia.GetRootControl()  # Desktop
for child in root.GetChildren():
    print(f"{child.ControlTypeName}: {child.Name}")
    
    # Recurse to children
    for grandchild in child.GetChildren():
        print(f"  {grandchild.ControlTypeName}: {grandchild.Name}")
```

### Check Scroll State

```python
import windows_mcp.uia as uia

control = uia.ControlFromHandle(window_handle)
scroll = control.GetPattern(uia.PatternId.ScrollPattern)

if scroll and scroll.VerticallyScrollable:
    percent = scroll.VerticalScrollPercent
    print(f"Scrolled {percent}%")
    
    if percent < 100:
        scroll.Scroll(uia.ScrollAmount.LargeIncrement, uia.ScrollAmount.NoAmount)
```

### Window Operations

```python
import windows_mcp.uia as uia

window = uia.WindowControl(Name="Calculator")
if window.Exists():
    pattern = window.GetPattern(uia.PatternId.WindowPattern)
    pattern.SetWindowVisualState(uia.WindowVisualState.Maximized)
```

## DPI Awareness

The wrapper automatically sets process DPI awareness to ensure coordinates match physical pixels:

```python
# Set at module load (core.py:1602)
uia.SetProcessDpiAwareness(ProcessDpiAwareness.PerMonitorDpiAware)
```

This prevents coordinate mismatches on scaled displays (125%, 150%, etc.).
